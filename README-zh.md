### 🚀 开始使用

1. **克隆此仓库**
   ```bash
   git clone https://github.com/ailingqu/h5games.git
   ```

2. **进入任意游戏目录**
   ```bash
   cd h5games/[游戏名称]
   ```

3. **在浏览器中打开 `index.html` 文件**
   ```bash
   open index.html
   ```

4. **开始游戏！**
