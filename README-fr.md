### 🚀 Commencer

1. **Clonez ce dépôt**
   ```bash
   git clone https://github.com/ailingqu/h5games.git
   ```

2. **Naviguez vers n'importe quel répertoire de jeu**
   ```bash
   cd h5games/[nom-du-jeu]
   ```

3. **Ouvrez le fichier `index.html` dans votre navigateur**
   ```bash
   open index.html
   ```

4. **Commencez à jouer !**
