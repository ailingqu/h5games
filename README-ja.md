### 🚀 はじめに

1. **このリポジトリをクローンする**
   ```bash
   git clone https://github.com/ailingqu/h5games.git
   ```

2. **任意のゲームディレクトリに移動する**
   ```bash
   cd h5games/[ゲーム名]
   ```

3. **ブラウザで `index.html` ファイルを開く**
   ```bash
   open index.html
   ```

4. **プレイ開始！**

            'event_category': 'games',
